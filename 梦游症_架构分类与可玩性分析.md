## 三、可玩性分析

### 3.1 设计意图 vs 实际体验

三条设计支柱的设计意图是好的，但当前状态下，**每条支柱都有断裂风险**：

**支柱 1：多样性是策略的根源**
- ✅ 设计意图正确：固定数值 + 多种定位 = 搭配决定成败
- ❌ 实际断裂：20只生物只有广度没有深度。每种定位2-3只选择看起来够用，但到后期玩家收集全了之后，策略空间不再增长——没有新的组合可发现
- ❌ 反应发现是黑箱：触发条件不透明，效果反馈不够强，玩家可能不知道化学反应发生了

**支柱 2：死亡让选择有重量**
- ✅ 设计意图正确：死亡不可逆 + 复活昂贵 = 每次出战有风险
- ❌ 实际断裂：惩罚过重可能形成**死亡螺旋**——死一只关键生物 → 缺少该定位 → 打不过后续关卡 → 没金币复活 → 更打不过
- ❌ 新手受伤继承是惩罚不是学习：第一次玩不了解受伤系统，结果主力生物连续受伤死亡，体验很差

**支柱 3：化学反应创造惊喜**
- ✅ 设计意图正确：混搭 > 纯阵营 = 探索奖励
- ❌ 实际断裂：当前9条反应中，很多还是数值叠加（军火交易 = 攻速+20%，古董交易 = 金币+5），不是行为变化
- ❌ 没有引导玩家发现反应的机制：图鉴没做，反应触发没有视觉爆点，玩家可能根本不知道反应存在

### 3.2 即时层乐趣分析（0-30秒）

**当前最大问题：玩家在战斗中几乎没有操作空间。**

生物固定不动 + 自动攻击最近目标 = 玩家在战斗中的唯一动作是"使用支援技能"。这比纯塔防的操作量还少——塔防至少可以在战斗中放新塔。

**潜在乐趣来源：**
- 看到生物HP告急 → 决定是否用治疗 → 这是即时决策的核心乐趣
- 化学反应触发 → 突然出现意想不到的效果 → 惊喜感
- 特殊敌人直冲核心 → 判断是否需要拦截 → 紧迫感

**但这些乐趣需要条件才能生效：**
- 治疗决策需要：治疗手段存在（当前缺失）、有足够信息判断（需要暂停/慢速）
- 反应惊喜需要：视觉爆点（当前缺失）、玩家知道反应存在（需要引导）
- 拦截紧迫感需要：反精英型生物存在（定义了但站位格系统缺失）

### 3.3 局层乐趣分析（3-10分钟）

**最大问题：惩罚机制过重，可能导致玩家"不敢玩"。**

阵容选择 + 受伤继承 + 死亡代价 = 理论上是风险博弈，但实际可能变成"带最强阵容保稳"的保守策略。如果每次出战都有可能永久失去一只生物，理性玩家会选择最小风险阵容，而不是尝试有趣的新搭配——这直接违反了"多样性是策略根源"的支柱。

**核心矛盾：支柱2（死亡有重量）和支柱1（多样性驱动）互相冲突。**

死亡惩罚越重 → 玩家越不敢尝试新搭配 → 多样性策略越少 → 支柱1失效。

### 3.4 长线层乐趣分析（小时-周）

**最大问题：只有收集广度，没有成长深度。**

20只生物收集全后 → 没有新生物可获取 → 策略空间不再扩展 → 游戏失去持续吸引力。

对比宝可梦的成功要素：
- 宝可梦有800+只 → 收集广度极大
- 宝可梦有升级/进化 → 成长深度极大
- 梦游症有20只 + 不升级 → 广度不够 + 深度为零

## 四、改进建议（按影响力排序）

### ★★★★★ 最高影响 — 不做这些游戏不够好玩

#### 建议 1：站位选择是唯一的微操，必须做到位

既然生物在战斗中不能移动，**战前部署阶段**就是玩家唯一有操作感的地方。这个阶段需要做深：

- 战前显示敌方波次预览（每波什么敌人、什么路线）
- 站位格布局可视化（前排/后排/侧翼/核心的位置关系和覆盖范围）
- 站位格之间有协同关系提示（放治疗型在后排→覆盖前排的治疗范围）
- 部署后可以"模拟预演"——看一眼站位覆盖路线的效果

**理由**: 站位塔防的乐趣 100% 来自战前规划。如果战前信息不够，玩家就是瞎放，没有策略感。

#### 建议 2：战斗中加入暂停/慢速按钮

生物不动意味着战斗中玩家几乎无事可做。加暂停和慢速（0.5x）让玩家能：
- 观察各生物HP状态
- 发现化学反应的触发
- 判断特殊敌人的路线
- 在暂停中使用治疗道具（暂停中可用道具，不暂停不可用 → 制造主动决策窗口）

**理由**: 这是把"被动观看"变成"主动观察"的关键。暂停不是作弊，是让玩家理解战场的工具。

#### 建议 3：化学反应必须有视觉爆点

当前化学反应触发后没有强视觉反馈。需要：
- 反应触发时：屏幕闪烁 + 粒子爆发 + 专属音效 + 2秒提示文字（"生化异变！"）
- 反应持续时：受影响生物有持续视觉标记（毒素光环/速度条纹/回血波纹）
- 反应首次触发时：图鉴自动解锁 + "新发现！"弹出

**理由**: 支柱3说"化学反应创造惊喜"。惊喜必须被看到才能被感受。如果触发后只是数值变了，玩家不知道发生了什么，那就没有惊喜。

#### 建议 4：平衡死亡惩罚 — 加"安全网"机制

当前的死亡螺旋风险太高。需要三道安全网：

**安全网 A — 新手保护期**: 前3-5局战斗中，生物死亡后自动复活（HP恢复50%），但受伤状态保留。这教会玩家"生物会死"但不惩罚学习过程。

**安全网 B — 24小时免费复活**: 死亡生物可以立即用金币+道具复活，也可以等24小时（现实时间）免费复活（HP恢复到满）。给玩家选择：急用付钱，不急免费。

**安全网 C — 死亡遗产**: 生物死亡后，它的站位格留下一个"遗志标记"，给站在这个位置的其他生物一个临时增益（比如+10%攻击持续3局）。这让死亡不只是损失，也有正面意义。

**理由**: 支柱2说"死亡让选择有重量"，但重量和绝望是两回事。重量是"我要慎重选择"，绝望是"我完了"。安全网把绝望变成重量。

#### 建议 5：给生物加1-2个装备槽 — 改行为不改数值

固定数值 + 不升级 = 没有成长感。解决方案：

- 每只生物有1-2个装备槽
- 装备改变**行为**而不是**数值**
- 例子：
  - "毒素涂层"：攻击附带3秒DOT（新增行为，不是+攻击力）
  - "视野透镜"：攻击范围变成"最近+最远同时攻击"（改变目标逻辑，不是+范围）
  - "急速齿轮"：攻击速度不变，但首次攻击延迟归零（改变节奏，不是+攻速）
  - "守护符"：HP不变，但濒死时有1秒无敌（新增行为，不是+HP）

**理由**: 装备不改数值 → 不违反支柱1。装备改行为 → 增加深度。每只生物可以有不同的装备选择 → 增加策略空间。这解决了"只有广度没有深度"的问题。

### ★★★★ 高影响 — 做了会显著提升体验

#### 建议 6：关卡设计教化学反应

早期关卡应该**强制引导**玩家发现化学反应：

- 关卡3: 只给 NATURE + TECH → 让玩家体验"生化异变"
- 关卡5: 只给 FAITH + MEMORY → 让玩家体验"回响"
- 关卡8: 给 3 派系 → 让玩家发现三派系混搭的可能性
- 每次首次触发新反应 → 图鉴解锁 + "新发现！"弹出 + 教程提示

**理由**: 玩家不会自己去试所有组合。关卡设计是最自然的引导方式。

#### 建议 7：受伤不降数值但要告诉玩家

企划书说受伤没有数值惩罚，但这个信息需要对玩家**极其明确**：

- 受伤时弹出提示："她的战斗能力没有下降——但她离死亡更近了"
- 轻伤→重伤时："危险！她随时可能死亡，但没有变弱——你需要保护她，不是因为她弱，是因为你不想失去她"
- 这个叙事把受伤从"我变弱了"（沮丧）变成"我还在战斗，但我需要你的帮助"（共情+紧迫）

**理由**: 受伤不降数值是好的设计，但如果玩家不知道这个规则，他们会直觉认为"受伤=变弱"，然后沮丧。必须用叙事明确告知。

#### 建议 8：扩展生物到30+只

20只太少。建议：
- 5阵营各4只 = 20只（基础）
- 5阵营各1只稀有 = 5只（Boss关奖励）
- 5只跨阵营融合生物 = 5只（特定化学反应解锁）
- 灰心岛5只特殊 = 5只（最高难度）
- 总计35只

**理由**: 收集游戏的持续吸引力 = 内容量。35只生物让图鉴完成感更有挑战，也让策略空间更大。

### ★★★ 中等影响 — 做了会更好但不做也能玩

#### 建议 9：波次间策略调整窗口

3-5波战斗之间加5秒喘息：
- 可以查看下一波预览
- 可以使用一次治疗道具（每波间限1次）
- 可以重新调整站位（限1次移动）

**理由**: 给玩家中途决策机会，避免战斗变成纯观看。

#### 建议 10：死亡生物的"回忆碎片"机制

死亡生物不是完全消失，而是留下"回忆碎片"：
- 回忆碎片可以嵌入城堡模块，提供该生物的部分能力
- 比如死了的治疗型生物 → 回忆碎片嵌入模块 → 城堡每波恢复5%HP
- 这保留了死亡生物的"一部分"，让死亡不那么绝对

**理由**: 符合"记忆"派系的世界观主题，也减少死亡的心理冲击。

#### 建议 11：关卡节点类型扩展

除了战斗节点，加：
- 商人节点（购买道具/装备）
- 遗迹节点（背景故事碎片 + 稀有道具）
- 事件节点（选择题 → 影响资源/好感）
- 休息节点（所有轻伤自动恢复）

**理由**: 地图不只是战斗走廊，休息和选择让节奏有起伏。

## 五、最终评估

### 可玩性评分

| 维度 | 当前 | 改进后 | 说明 |
|------|------|--------|------|
| 即时层乐趣 | ★★☆☆☆ | ★★★★☆ | 加暂停+视觉爆点+治疗窗口后大幅提升 |
| 局层乐趣 | ★★☆☆☆ | ★★★★☆ | 加战前部署+保护期+安全网后提升 |
| 长线乐趣 | ★★☆☆☆ | ★★★☆☆ | 需要扩展内容量(30+只)才能支撑 |
| 策略深度 | ★★★☆☆ | ★★★★☆ | 装备槽+站位规划+反应引导后提升 |

| **综合** | **★★☆☆☆** | **★★★★☆** | **核心设计意图正确，关键缺口补上后可玩性很高** |

### 一句话结论

> 梦游症的3条设计支柱方向正确——多样性策略、死亡风险、化学惊喜都是好想法。但当前实现缺失了6个P0级核心机制（站位格、生物HP、受伤阶段、死亡、敌人打生物、战前部署），导致游戏还是纯塔防而不是企划书描述的站位塔防出怪。补齐这些缺口 + 加上5个最高影响改进建议（战前部署深度、暂停/慢速、反应视觉爆点、死亡安全网、装备槽），游戏可玩性可以从★★提升到★★★★。
